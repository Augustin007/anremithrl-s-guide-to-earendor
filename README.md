#### Timeline
Large scale timeline. Contains one folder for each age, labelled by the number at the beginning, containing only one file, the master file for the age. Contains Ages Past and Campaign Chronology as directories for the ages and the campaigns. 

### Campaigns
This contains overviews of each campaign in the world. While the base directory of it is in `World/Timeline/Campaign Chronology.md`, I consider the Campaigns to not be part of the lore. 

### Meta
Contains meta-information about the vault. Such as mechanics. Images. Decks of Cards, dataviews and metadata, unfinished files, and Standard Formats.
