
# Earendor

[Link to Website](https://augustin007.github.io/anremithrl-s-guide-to-earendor/)


Hello, this is the code and data for the Earendor Wiki, Anremithrl's Guide to Earendor

This works by having an obsidian vault, and a pair of scripts to scrape the vault and reformat it correctly, and finally having mkdocs use the reformatted version to create a website.
